# page-resizable 可拖动大小页面

可拖动页面大小页面组件，支持页面水平方式两栏拖拽大小。

## 基本使用

::: demo
page-resizable/page-resizable1
:::

## 组件 API

### Attributes 属性

| 参数             | 说明       | 类型      | 可选值          | 默认值  |
| -------------- | -------- | ------- | ------------ | ---- |
| left-min-width | 左侧区域最小宽度 | Number  | -            | -    |
| is-draggable   | 是否可拖拽    | Boolean | true / false | true |

### Methods 方法

| 方法名 | 说明  | 参数  | 返回值 |
| --- | --- | --- | --- |
|     |     |     |     |

### Events 事件

| 事件名 | 说明  | 参数  | 返回值 |
| --- | --- | --- | --- |
|     |     |     |     |

### Slots 插槽

| 插槽名   | 说明   | 参数  |
| ----- | ---- | --- |
| left  | 左侧区域 |     |
| right | 右侧区域 |     |
