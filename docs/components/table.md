# table 列表

基于JSON Schema 的列表组件，组 element-plus 中 el-table 的深度封装，快速实现列表。只具有根据 JSON Schema 和 UISchema 展示列表的能力，不具备表格卡片的能力。

## 基本使用

::: demo
table/table1
:::
